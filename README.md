# Telegram Bot

Telegram bot that submits feedback into the API.

## Setup

```bash
npm install
cp .env.example .env
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
npm run start
```

## Environment

- `TELEGRAM_BOT_TOKEN`: Telegram bot token from BotFather
- `API_BASE_URL`: Base URL for the deployed API, including `/api`
